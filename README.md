# Pac-Man Arcade
A classic Pac-Man game built with HTML, CSS, and Vanilla JavaScript.

*This project was created during the Vibecode Arena challenge by HackerEarth.*

## Features
- Classic grid-based arcade action
- 4 Unique Ghosts that chase you
- Power Pellets to turn the tables and eat the ghosts
- Step-by-step classic movement
- Fully working UI with live Score and Lives tracker
- Nostalgic Sound Effects built purely with the Web Audio API

## How to Play
1. Click the **Start Game** button.
2. Use your **Arrow Keys** to steer Pac-Man around the board. Pac-Man moves exactly one tile step per key press.
3. **Eat all the small white dots** to win the game. Each dot is worth 10 points.
4. **Avoid the Ghosts!** If a ghost touches you, you will lose a life. Look out for:
   - **Blinky** (Red)
   - **Pinky** (Pink)
   - **Inky** (Cyan)
   - **Clyde** (Orange)
5. **Power Mode**: Eat the large glowing power pellets in the corners to turn the ghosts blue. While they are blue, you can eat them for a massive 200 points!

## Technical Details
- Built in a single `index.html` file
- No external game frameworks (pure Canvas rendering)
- No external sound files (synthesized sounds using AudioContext)

Enjoy the game!
